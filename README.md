PROJECT TITLE 
# AMAZON PRIME DATA VIDEO DATA ANALYSIS

PROJECT  DESCRIPTION
This project analyzes  Amazon Prime content data to understand trends in movies and TV shows.
The goal is to explore:
- Content distribution
- Popular genres
- Release trends over the years

  DATASET INFORMATION
  The dataset contains information about Amazon Prime movies and TV shows.
  - Title
  - Release Year
  - Duration
  - Country

    TOOL USED
    - Python
    - Pandas
    - Matplotlib
    - Seaborn
    - Plotly

     PROCESS
    - Data import(file import)
    -  Data cleaning
    -  Handling missing value
    -  Data Analysis
    -  EDA
    -  Data Visualisation
